DOM

JSONP

JSONP是一個非官方的協議，它允許在伺服器端集成Script tags返回至客戶端，通過javascript callback的形式實現跨域存取

Server X 
			
<script type="text/javascript" 
src="http://server2.example.com/RetrieveUser?UserName=張三&jsonp=parseResponse">
 </script>

Server X Callback 
parseResponse({
 　"name":"張三",
 　"password":"123456",
 　"department":"技術部",
 　"sex":"男",
 　"old":30
});
